Blog de viajes

Este es un proyecto de un Blog de Viajes realizado casi completamente con Bootstrap.
En la pagina principal cuenta con un slider, un navbar fijo que baja junto con el usuario, un breve comentario de las entradas del blog con una imagen y el link y una barra lateral donde se mencionan las entradas anteriores.

Este es un proyecto de una pagina Web para un emprendimiento realizado con CSS y Bootstrap.

La segunda imagen corresponde a la historia del emprendimiento con imagenes y texto, mientras que en la tercera se ve la galeria de imagenes de los productos en donde se puede acceder a cada uno de ellos.
